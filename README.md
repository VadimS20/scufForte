# scufForte

## Install run client and server
```
git clone https://github.com/VadimS20/scufForte.git
```

Server
```sh
cd scufford
mkdir build
cd build
cmake ..
make
./SCUFFORD
```
Client
```sh
cd client
mkdir build
cd build
cmake ..
make
./CLIENT
```
## Flags

Server

-f "path to your file" #you can use xml or fboot

-p "port" #port for connection default: 61499

Client

-f "path to your file" #you can use xml or fboot

-p "port" #port for connection default: 61499
